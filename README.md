
# Bengaluru House Price Prediction

End-to-end beginner data science project using the Bengaluru house price dataset.

## What this project does
- Loads and cleans the Kaggle Bengaluru_House_Data.csv using pandas and NumPy.
- Explores the data with visualizations (histograms, scatter plots, bar charts).
- Trains a basic linear regression model to predict house prices.
- Organized for learning and as a portfolio project.
